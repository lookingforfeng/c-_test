# c++学习中用到的测试demo
## 1、friend_func_test
+ 用于测试友元函数和友元类的使用
+ 友元主要应用于在c++类的外面需要访问类的私有成员的情况，例如序列化输出函数
+ 它破坏了类的封装性和隐藏性，使得非成员函数可以访问类的私有成员
## 2、func_point_test
+ 用于测试函数指针的申明和使用
## 3、extern_test
+ 用于测试extern关键字的使用，extern的意思就是函数的实现在另外一个文件中，编译器自己去找。
